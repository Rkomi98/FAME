Setup Rapido

Inizia in pochi minuti

1. Clona il Repo
git clone https://github.com/tuonome/FAME.git
cd FAME/Soluzione
2. Setup Ambiente
python -m venv FAME
source FAME/bin/activate
pip install -r requirements.txt
3. Ottieni API Key
  • Google Gemini (Gratis)
  • OpenAI (Pay-per-use)
4. Avvia l'App
python app.py

Visita: http://localhost:5000
